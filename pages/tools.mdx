# Tools for Ships

While the game is designed for Grant Ships to enjoy maximum flexibility for how they want to runm their ships, the designers of the game are seasoned DAO operators and have gained experience in using many of the tools DAOs can use in order to colloboarate. We offer these as possibilities and offer a brief description of of how it can be used and links to their websites. We do not offer support for any of these tools.

- Gitcoin Grant Stack
    - Allo
    - Hypercerts
    - Gitcoin Passport
- Hats Protocol
- Jokerace
- DAOhaus
- Discourse
